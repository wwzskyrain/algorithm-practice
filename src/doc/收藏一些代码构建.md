# 代码构建

## 所有子集的和

```java
public class AllSetSum {
    private int[] A;

    public Set[] f() {
        int m = A.length / 2;
        Set[] dp = new int[m + 1];
        for (int num : A) {
            for (int i = m; i >= 0; i--) {
                // 随着越来越多的A元素加入，dp[高段位]才会主键有值
                // 到最后才能到dp[m]
                for (int sum : dp[i - 1]) {
                    dp[i].add(sum + num);
                }
            }
        }
    }
}
```
